# Material Filter

Filter stored items by the material they are made from — in stockpiles, bills, and apparel policies.

A RimWorld 1.6 mod by Kami Katze, forked by CheaterEater. Fork of [Kami Katze's Material Filter](https://steamcommunity.com/sharedfiles/filedetails/?id=1541305730), rewritten to use Harmony (no HugsLib dependency) with performance and QoL improvements.

## Notes

- The original mod's save key format is preserved, so existing material filter settings remain compatible
- The popup now renders native in-game material icons with search.
